# LightStudio Pro — תוכנת תכנון תאורה בעברית

## English

Professional lighting design calculator with a **fully Hebrew** interface. Built for Hebrew-speaking lighting designers, architects, and electricians.

- Room lighting calculations using the CIE Lumen Method
- Multi-layer lighting design (general, functional, ambient)
- Visual fixture placement on a 2D floor plan
- Compliance checking against EN 12464-1 / IESNA standards
- Energy and quality scoring

### Install & Run

```bash
pip install lighting-designer
ldesign
```

## עברית

תוכנה מקצועית לתכנון תאורה — **ממשק מלא בעברית**.

### יכולות

- **חישוב תאורה** לפי שיטת הלומן (CIE Lumen Method) עם מקדמי ניצולת ותחזוקה
- **שכבות תאורה** — תאורה כללית, פונקציונלית ואווירה, כולל חלוקת לומן ואחוזים
- **תצוגת חדר** — ציור דו-ממדי של החדר עם מיקומי גופי תאורה (פאנלים, ספוטים, רצועות LED)
- **16 סוגי חדרים** — סלון, משרד, מטבח, כיתה, חנות, מסעדה, בית חולים ועוד
- **בדיקת תקן** — עמידה בדרישות EN 12464-1 ו-IESNA
- **ציון איכות** — דירוג 0–100 על בסיס רמת לוקס, איזון שכבות ויעילות אנרגטית
- **צריכת אנרגיה** — הערכת צריכה שנתית ב-kWh וצפיפות הספק (W/m²)
